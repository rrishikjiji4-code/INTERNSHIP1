⚠ Sensor Diagnostics Guide
01.
Secure Host (HTTPS) Required: Acceleration APIs will fail silently in non-secure HTTP connections. Verify your address bar starts with https:// or localhost.
02.
Apple iOS Permission Safari Settings: If prompts fail, navigate to Settings › Safari › Motion & Orientation Access and toggle it ON. Reload the page.
03.
Chrome / Android Device Sensor Access: Android devices usually auto-allow, but if blocked, go to Settings › Site Settings › Motion Sensors and set to Allowed.
